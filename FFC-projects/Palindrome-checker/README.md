Palindrome project from FreeCodeCamp

-The task is to write a function, checking whether the string is a palindrome.
The function should:
1) remove all non-alphanumeric characters.
2) convert the string to lowercase.
3) compare the clean string to it's reversed partner.

If they match, the function returns true, and if not, returns false.

-- How to run the code:
- use Node to run the script and type node palindrome.js
